Conceptos

Window es el objeto que representa la ventana abierta del navegador, es decir la página que tiene cargada nuestro navegador.

closed

Devuelve true en caso de que la ventana a la que se está haciendo referencia con el objeto (por defecto la actual) esté cerrada.

console

Da acceso a la consola del navegador, puede usarse directamente console o acceder mediante la propiedad de Window.

document

Da acceso al objeto documento del navegador, puede usarse directamente document o acceder mediante la propeidad de Window. El objeto documento es aquel se crea a partir del documento HTML generado ¡Es como si la página HTML fuera un objeto!

frameElement

Devuelve el frame de la ventana en caso de que tenga, en caso contrario da null. Es una propiedad de solo lectura y también se puede acceder mediante frameElement.

frames

Devuelve todos los objetos Window que tenga una Window en forma de array.

history

Da acceso a las paginas visitadas por el usuario desde esa Window. Es una propiedad de solo lectura y también se puede acceder con history.

innerHeight

Devuelve la altura de la ventana. Se puede acceder mediante innerHeight también.

innerWidth

Devuelve la anchura de la ventana. Se puede acceder mediante innerWidth también.

length

Devuelve la cantidad de frames que tiene una ventana.

localStorage

Permite guardar una clave/valor en el navegador sin fecha de expiración. Se puede acceder mediante localStorage también.

location

Devuelve el objeto location el cual contiene información sobre la dirección actual. Se puede acceder también mediante location

Funcion Parametros y definicion

EL OBJETO DOCUMENTO

El objeto documento es una propiedad del objeto de ventana. Cuando el documento HTML se carga en un navegador se convierte en un objeto del documento siendo el nodo raiz del documento HTML.

activeElementpropiedad que devuelve el elemento al que se le esta haciendo focus.

baseURIPropiedad que devuelve l'URI base de un documento. URI es el identificador uniforme de recursos, un texto corto que identifica sin error cualquier recurso accesible de la red.

bodyPropiedad que establece o devuelve el body de un documento

characterSetPropiedad que devuelve la codificación de los carácteres para un documento.

cookiePropiedad que establece o devuelve una lista separada por ";" con pares (clave => valor) de las "galletas" o cookies del documento.

defaultViewPropiedad que devuelve el objeto ventana del documento.

designModePropiedad que establece o devuelve si el documento es editable.

doctypePropiedad que devuelve el tipo de documento de un documento.

addEventListener()Método que adjunta un controlador de eventos a un documento. Los controladores de eventos llevan el mismo nombre que el evento al que se asocian precedidos del prefijo on.

adoptNode()metodo que adopta un nodo de otro documento. Un nodo es cualquier etiqueta del body, como por ejemplo un <p> o incluso el <body>. Con JavaScript podemos modificar estas etiquetas.

close()Metodo que cierra una ventana previamente abierta con el metodo open()

createAttribute()Metodo que crea un atributo y devuelve el atributo como objeto de Atrr.

createComment()Metodo que crea un comentario y devuelve el nodo de comentario

createDocumentFragment()Metodo que crea un nodo fuera de la pantalla, para crear un fragmento de un nuevo documento y poder insertarlo en cualquier otro documento. También se puede utilizar para cambiar partes o añadir y volveras a añadir en el nuevo documento.

createElement()Metodo que crea un elemento nodo.

createEvent()Metodo que crea un evento de objeto.

createTextNode()Metodo que crea un texto de nodo
